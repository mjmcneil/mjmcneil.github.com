ENAC

In the fall of 2012 I had the opportunity to go work in Toulouse, France at Ecole Nationale de l'Aviation Civile as a research intern. While I was there I work on a project to remodel a prototype interface for their open source software platform Paparazzi used to control unmanned aerial vehicles. I analyzed the current user functions and structure of Paparazzi including going out into the field to observe user interactions. From these observations I designed the user interactions and the user interface for a prototype system my lab was developing. For the prototype system I created all of the SVG graphics and then implemented these components with the IStar and Paparazzi framework.

Teledyne Dalsa

In the winter of 2012 I worked at Teledyne Dalsa in Waterloo, a company that specializes in digital imaging and semiconductors. While at Dalsa I worked as a web developer under corporate communications. There I built upon their database functionality including transitioning static language content to be dynamically sourced from database content or global resource files and developing content authoring tools to add, edit, and publish website content. I also transferred an in-house frame rate calculation Excel algorithm into a responsive web page controlled by VB code behind dynamic look up tables.

While working at Dalsa, I had the opportunity to work at the Kitchener-Waterloo United Way one day a week to help them out with their technical operations in the office. Over the course of the work term I was able to develop a project-team matching system for them to accommodate their local annual event "The Day of Caring". The system I created allowed perspective projects and volunteer team to sign up online and then matched up based on group size or interest area by a United Way Employee. The final piece of the system allowed synchronized group emails out to projects and volunteer teams simultaneously. Overall this system saved lots of paper work and confusion by having one central system.

IFDS

In the summer of 2011 I worked downtown Toronto at the backend financial company Internation Financial Data Services (IFDS). I was a part of the research and development teams who's role was to introduce new innovative technologies into the office work space. While at IFDS I worked on numerous projects including a Work Queue Assignment System and the Xbox 360 Kinect sensor. Initially I used the Xbox Kinect to register user movements utilising C# with open source software packages OpenNI and PrimeSense-NITE to control the Work Queue Assignment. After I created a functioning backend structure to interface with existing company software using SQL and PHP, I transitioned the Work Queue Assignment system from a Kinect application to a more practical iPad web application. My last project at IFDS was to created user gesture with the Kinect which would cause less user fatigue to control existing company software.
